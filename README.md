### Introdução
Este é o arquivo contendo o algoritmo de PRocessamento de Linguagem Natural, desenvolvido para a Disciplina de Inteligência Artificial, ministrada pelo professor Yuri. 

### Clone do Projeto:

```
$ git clone https://github.com/franciscoffavaro/nlp.git
```

### Instalação de Dependências:
Na pasta Raiz do Projeto:
```
$ npm install
```

### Execução:
```
$ node src/main.js
```
